# VibeLink

A cross-platform connectivity hub for haptic feedback devices with protocol support.

## Overview

VibeLink provides a unified interface for controlling Bluetooth LE peripherals through various input methods.
Designed for real-time responsiveness and flexible integration with external applications.

## Features

### Multi-Protocol Support
- **Bluetooth GATT** - Standard BLE device communication compatible with name brand devices
- **Bluetooth ADV** - Simplified BLE advertisement-based protocol for compatible hardware (Linux only)
- **OSC Control** - Network-based control with simple parameter mapping
- **Remote Control** - Long-distance remote control

## Requirements

- Bluetooth 4.0+ adapter
- Linux/windows
- (Optional) ngrok auth token for remote control features
